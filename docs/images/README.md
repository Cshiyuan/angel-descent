# 图片资源说明

本目录存放 README.md 中使用的展示图片。

## 图片列表

- `qrcode.jpg` - 小游戏二维码（需要手动添加）
- `game-start.jpg` - 游戏开始界面
- `scene-dawn.jpg` - 朝霞天界场景
- `scene-cloud.jpg` - 云海天界场景
- `scene-thunder.jpg` - 雷音天界场景
- `scene-earth.jpg` - 凡间边界场景
- `life-fruit.jpg` - 生命果实系统
- `frozen.jpg` - 冰冻效果展示

## 添加二维码

请将微信小游戏二维码保存为 `qrcode.jpg` 并放置在此目录。

建议尺寸：600x600 像素或更大
